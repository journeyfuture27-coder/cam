function drawSector(map, centerLngLat, direction, rangeMeters, coneAngleDeg = 40) { const halfAngle = coneAngleDeg / 2; // Преобразуем навигационный азимут в математический function getMathAngle(azimuth) { return (90 - azimuth + 360) % 360; } const leftMathAngle = getMathAngle(direction - halfAngle); const rightMathAngle = getMathAngle(direction + halfAngle); // Три точки: вершина, левая точка, правая точка const leftPoint = getPointAtDistanceMath(centerLngLat[0], centerLngLat[1], leftMathAngle, rangeMeters); const rightPoint = getPointAtDistanceMath(centerLngLat[0], centerLngLat[1], rightMathAngle, rangeMeters); const points = [ [centerLngLat[0], centerLngLat[1]], [leftPoint[0], leftPoint[1]], [rightPoint[0], rightPoint[1]], [centerLngLat[0], centerLngLat[1]] // замыкаем ]; const geometry = { type: 'Polygon', coordinates: [points] }; try { const sector = new ymaps3.YMapFeature({ geometry: geometry, style: { fillColor: 'rgba(200, 100, 50, 0.35)', strokeColor: '#cc6633', strokeWidth: 1.5 } }); map.addChild(sector); return sector; } catch (error) { console.error('Ошибка рисования сектора:', error); return null; } }
